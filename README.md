- **Upload Your PFP** - Upload any Doginal Dog image to become the player character

- **Test PFP Included** - Use `test-pfp.webp` to test the upload feature

1. Open `index.html` in a browser
2. Click "Upload PFP" to upload your Doginal Dog image (or use the included test PFP)
3. Click "Play as Default" to try with the built-in pixel dog

- Work best when square
